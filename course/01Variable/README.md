## 变量

* python中的变量赋值不需要类型声明
* = 用于给变量赋值
* 整数型
  * count = 1
* 浮点型
  * pi = 3.14
* 字符型
  * sir = 'huang'
* 五种标准数据类型
  * Numbers
    * Int 有符号整数
    * Float 浮点数
    * Complex 复数
    * Bool 布尔类型（True / False）
  * String
  * List
  * Tuple
  * Dictionary
